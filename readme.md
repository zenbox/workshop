## Workshop

Wir entwickeln im Laufe der Woche verschiedene React- und Next.js Projekte _from scratch_. Wir schreiben alle Codes in Visual Studio Code.
Die Ergebnisse werden hier veröffentlicht und stehen Ihnen anschließend unter der Common Creative Lizenz CC BY-NC-SA zum Download zur Verfügung.

## Schulungsziele I

Nach dem Schulungsbesuch sind Sie in der Lage, Websites von der interaktiven App bis zur statischen Landing-Page mit dem Stack aus React und Next.js zu erstellen. Sie verstehen die entsprechenden Grundelemente, wie Komponenten, States, JSX und Hooks. Außerdem sind Sie mit Libraries aus dem React-Universum, wie z.B. für Styling, API-Kommunikation oder Formulare, vertraut.

## Schulungsziele II

Nach der Aufbau-Schulung beherrschen Sie den Umgang von React-Hooks, schreiben mit Jest automatisierte Tests und beherrschen die Tools zur Performanceoptimierung.

## React und Next.js II - Validierung, Testen und Performance

Im zweiten Teil der React und next.js Schulung werden Tests mit Jest und react-testing-library behandelt. Außerdem werden die Möglichkeiten zur Performanceoptimierung in React und Next.js vorgestellt. Ein weiterer Schwerpunkt ist das Thema "Hooks", die in React die Verwendung von State und Lifecycle-Methoden in funktionalen Komponenten ermöglichen.

Die Schulung ist als Aufbau-Schulung konzipiert und setzt die Inhalte der Grundlagen-Schulung voraus.

-   **Testen**

    -   Testen von JavaScript-Funktionen mit Jest
    -   End-to-End-Tests mit JavaScript
    -   Testen von React-Komponenten mit react-testing-library
    -   Snapshot-Tests

-   **Externe und eigene Hooks**

    -   Anwendungsgebiete von Hooks
    -   Definieren eigener Hooks
    -   Verwenden vorgefertigter Hooks

-   **Context**

    -   Teilen von Daten über einen ganzen Komponentenbaum hinweg
    -   das Problem des "prop drilling"

-   **APIs/Daten aus React abfragen (Teil 2)**

    -   ReST-APIs mit "fetch" abfragen
    -   Die "TanStack Query" Library

-   **Formulare und Validierung**

    -   Komplexe Formulare validieren
    -   Das Paket "react-hook-form"
    -   Ein Blick in das Paket "formik"

-   **Performanceoptimierung**
    -   React Developer Tools Profiler
    -   React.memo (memoisierte Komponentenrenderings)
    -   Callback-Hook (memoisierte Callbacks / Eventhandler)
    -   Memo-Hook (memoisierte abgeleitete Werte)
